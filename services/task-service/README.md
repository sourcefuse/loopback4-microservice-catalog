## Overview

A reusable, customizable and workflow based task service which creates some actionable tasks based upon various events happening via different microservices in a distributed system.

To get started with a basic implementation of this service, see [/sandbox/task-example](https://github.com/sourcefuse/loopback4-microservice-catalog/tree/master/sandbox/task-example).
