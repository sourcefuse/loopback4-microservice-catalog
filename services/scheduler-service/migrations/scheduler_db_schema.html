Table subscriptions

Subscribe is a collection of all calendar entries that a user has added to their list.
You can use it to add and remove existing calendars to/from the users’ list.
You also use it to retrieve and set the values of user-specific calendar properties, such as default reminders.
Another example is foreground color, since different users can have different colors set for the same calendar.

IdxField NameData TypeDescription
* id uuid DEFAULT uuid_generate_v1()
  access_role varchar DEFAULT 'reader'::character varying The effective access role that the user has on the calendar. Possible values are:
"freeBusyReader" - Provides read access to free/busy information.
"reader" - Provides read access to the calendar. Private events will appear to users with reader access, but event details will be hidden.
"writer" - Provides read and write access to the calendar. Private events will appear to users with writer access, and event details will be visible.
"owner" - Provides ownership of the calendar. This role has all of the permissions of the writer role with the additional ability to assign acess to other users.

The owners of a calendar can share the calendar by giving access to other users.
  bg_color varchar The scheduler color of the calendar in the hexadecimal format "#0088aa".
* calendar_id uuid
  created_by varchar User/Person Email Id.
* created_on timestamptz DEFAULT CURRENT_TIMESTAMP
  default_reminders json The default reminders that the user has for this calendar.
  deleted boolean DEFAULT false For doing soft delete operation on user calendard. Read-only. Optional. The default is False.
  ext_id varchar
  ext_metadata jsonb
  fg_color varchar The foreground color of the calendar in the hexadecimal format "#ffffff".
* identifier varchar User/Person Email Id.
  is_hidden boolean DEFAULT false
  is_primary boolean DEFAULT false
  modified_by varchar User/Person Email Id.
* modified_on timestamptz DEFAULT CURRENT_TIMESTAMP
  notification_settings json The notifications that the user is receiving for this calendar.
