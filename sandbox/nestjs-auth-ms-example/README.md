# NestJS - Auth Service Example

## Description

This is an example NestJS Application that uses ARC's (aka SourceLoop) pre-built microservice for authentication.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
